Talk: Abusing Intended Features and Bypassing Facial Recognition Check

Abstract:
This presentation analyzes vulnerabilities in facial recognition technology, focusing on how deepfake images can bypass security measures. It discusses an AI app that maps user faces from group photos for automatic categorization and retrieval, with a registration process involving mobile/email sign-up, OTP verification, and live face detection.

A critical flaw is identified in the face detection feature, which allows users to import pictures for verification. The system's overly simplistic criteria—only requiring the face to be human and looking straight—fail to distinguish between live faces and deepfakes. This oversight makes the system vulnerable to manipulation, enabling bypass through deepfake images that meet basic requirements. The author shares firsthand testing, showing how deepfakes can successfully bypass the system checks.

Ishwar Kumar

Ishwar Kumar is an aspiring security researcher exploring vulnerabilities in cutting-edge technologies. His research includes bypassing security mechanisms in facial recognition systems using deepfake techniques, highlighting critical gaps in current implementations.

Ishwar is passionate about uncovering security flaws and creating awareness about potential threats in modern applications. His findings aim to inspire developers and organizations to strengthen defenses against emerging vulnerabilities.
